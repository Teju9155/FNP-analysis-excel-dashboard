# Sales Analysis Dashboard (Excel)

## Overview
This project presents an interactive Sales Analysis Dashboard built in Excel to track and analyze business performance across orders, revenue, customers, products, and delivery timelines.

The dashboard is designed to provide a clear, single-view summary of key metrics and trends, enabling data-driven decision making.

---

## Key Metrics
- Total Orders: 1000  
- Total Revenue: ₹35,20,984  
- Average Customer Spend: ₹3,520.98  
- Average Order-to-Delivery Time: 5.53 days  

---

## Features
- Dynamic filtering using slicers (Order Date, Delivery Date, Occasion)
- Fully interactive dashboard with connected visuals
- Clean, single-screen layout for easy interpretation

---

## Tools & Techniques Used
- Power Query  
  - Data cleaning and transformation  
  - Handling raw dataset for analysis  

- Power Pivot  
  - Data modeling  
  - Creating relationships between tables  
  - Calculated measures (KPIs)  

- Pivot Tables & Charts  
  - Aggregation and summarization  
  - Visual representation of insights  

---

## Dashboard Insights

### Revenue Analysis
- Revenue is driven significantly by key occasions such as Anniversary and Raksha Bandhan  
- Certain occasions like Diwali and Valentine’s Day show comparatively lower performance  

### Category Performance
- Top contributing categories: Colors, Sweets, Soft Toys  
- Lower performing categories: Mugs, Plants  

### Geographic Trends
- Top cities by orders include Imphal, Kavali, and Dhanbad  
- Indicates region-specific demand concentration  

### Time-Based Trends
- Revenue peaks observed in February and August  
- Weekly trends show stronger sales on Tuesdays and Sundays  

### Product Performance
- Top products: Magnam Set, Quia Gift, Dolores Gift  
- These products contribute significantly to total revenue
